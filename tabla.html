¿Que es una tabla html y para que sirve?

Las tablas HTML están pensadas para utilizarse con datos tabulados. Por desgracia, mucha gente utiliza las tablas HTML para hacer compaginaciones de páginas web. Por ejemplo, una fila para contener la cabecera, una fila para contener las columnas de contenido, una fila para contener el pie de página, etc.

Etiquetas basicas

TABLE: crear tabla

TR: crear fila

TH: define celda de encabezado de la tabla

TD: crea celda

CAPTION: define el titulo de la tabla

COLGROUP: especifica un grupo de una o mas columnas para ampliar formato

COL: especifica las propiedades de una columna de las columnas definidas de un elemento colgroup

THEAD: define el encabezado de la tabla

TBODY: define el cuerpo de la tabla

TFOOT: define el pie de tabla
